# Security Policy

## Supported Versions

PulseLine Card is a small Home Assistant custom card. Security updates are provided for the latest released version only.

| Version | Supported |
| ------- | --------- |
| Latest  | Yes       |
| Older releases | No |

## Reporting a Vulnerability

If you believe you have found a security vulnerability in PulseLine Card, please report it privately instead of opening a public issue.

Use GitHub's private vulnerability reporting feature if it is available on this repository.

If private reporting is not available, please open a GitHub issue with a brief description that does not include sensitive details, and I will coordinate a private follow up.

Please include:

- A short description of the issue
- Steps to reproduce, if safe to share
- The affected version
- Any relevant Home Assistant or browser context

I will try to acknowledge reports within 7 days. If the issue is accepted, I will work on a fix and publish a release as appropriate.
